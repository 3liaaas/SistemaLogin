# Sistema de Login para Administração
Este projeto é um sistema de login para funcionários de alta liderança ou administrados de empresas, com um limite de tentativas e bloqueo da conta, caso haja ultrapassagem do valor limite.
## Requisitos
- IDE focada em Java, como Intellij ou Eclipse
- Java 25
- JDK

Não há necessidade de instalar alguma ferramenta de terceiros.
